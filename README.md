Explanation of code :
Objective
The main objective of this project is to create an interactive number guessing game using Java.
The program generates a random number between 1 and 100, and the user has to guess the correct number within a limited number of attempts. The project helps in understanding Java programming concepts like loops, conditions, random number generation, user input handling, and score calculation.
Steps Performed
Import Required Packages
Scanner is used to take user input.
Random is used to generate a random number.
Initialize Game Settings
Total score and number of rounds are initialized.
The game is set to run for multiple rounds.
Generate Random Number
The system generates a random number between 1 and 100 for each round.
Take User Input
The user enters a guessed number through the keyboard.
Compare Guess with Generated Number
If the guess is correct → user wins the round.
If the guess is higher → program displays “Too High”.
If the guess is lower → program displays “Too Low”.
Limit Attempts
The player gets only a fixed number of attempts to guess the number.
Calculate Score
Points are awarded based on the remaining attempts after guessing correctly.
Display Final Result
After all rounds, the total score is displayed.
